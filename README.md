#Windows Event Log Analyzer
Small Python tool to parse Windows Security Event Logs (exported as XML)
and extract selected Event IDs for analysis.

## Features
- Parses exported Windows Event Viewer XML files
- Filters specific Security Event IDs
- Outputs structured data
- Includes basic pytest coverage

## Tech stack
- Python 3
- Linux (WSL)
- pytest
- xml.etree.ElementTree
